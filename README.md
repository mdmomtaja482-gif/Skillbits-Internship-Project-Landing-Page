## ✨ Key Features

- **Interactive Hero Section** — Dynamic gradient text, stats counters, and CTA buttons with ripple effects.
- **Custom UI Dashboard Mockup** — A beautiful interactive learning card showing live progress bars, active session indicator, and instructor avatars (replaces static images).
- **Course Cards Gallery** — 4+ popular programs displayed with engaging hover animations and click-to-enroll interactions.
- **Micro-Interactions** — Button ripple effects, stat number scaling on hover, smooth page scroll, and real-time toast notifications.
- **Fully Responsive** — Optimized for desktops, tablets, and mobile devices using flexbox and media queries.
- **Newsletter / Lead Capture** — Email subscription with validation and instant feedback.
- **No External Dependencies** — Pure vanilla code — no frameworks or libraries (except FontAwesome for icons).
